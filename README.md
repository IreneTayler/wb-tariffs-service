# WB Tariffs Service

This is a FastAPI project for handling tariffs information. The service fetches tariffs data and stores it in a PostgreSQL database. It provides an API to create and retrieve tariffs.

## Features
- Fetch tariffs data from an external source.
- Store tariffs data in a PostgreSQL database.
- API endpoints to create, retrieve, and manage tariffs.
- Dockerized for easy deployment and testing.

## Requirements
- Python 3.8+
- PostgreSQL 13+
- Docker (optional)
